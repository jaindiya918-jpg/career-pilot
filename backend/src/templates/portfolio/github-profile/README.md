# GitHub Profile Portfolio Theme

A professional, high-fidelity developer portfolio template mirroring the structured and clean aesthetic of a GitHub profile. Built for the CareerPilot platform.

---

## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | Semantic HTML with `{{PLACEHOLDER}}` tokens, Left Sidebar, Right Area |
| `style.css` | GitHub-style dark-mode colors, contribution heatmap, repository cards |
| `script.js` | Placeholder replacement, contribution calendar builder, repository dot color mappings, README markdown rendering |

---

## 🎨 Design System

### Colour Palette

| Variable | Value | Usage |
|----------|-------|-------|
| `--bg-main` | `#0D1117` | GitHub Dark main body background |
| `--bg-canvas` | `#010409` | Dark canvas background behind headers |
| `--bg-card` | `#161B22` | Dark repository/content card background |
| `--border-color`| `#30363D` | Soft slate border color for outlines |
| `--text-main` | `#E6EDF3` | Primary body text and headers |
| `--text-muted` | `#848D97` | Muted descriptions, stats, secondary links |
| `--accent-blue` | `#58A6FF` | Link hover states and accent blue icons |
| `--accent-green`| `#238636` | High-emphasis green actions (e.g. follow button) |

### Green Contribution Heatmap Palette

| Variable | Value | Level |
|----------|-------|-------|
| `--contrib-bg` | `#161b22` | Level 0: No contributions |
| `--contrib-l1` | `#0e4429` | Level 1: Low contributions |
| `--contrib-l2` | `#006d32` | Level 2: Medium-low contributions |
| `--contrib-l3` | `#26a641` | Level 3: Medium-high contributions |
| `--contrib-l4` | `#39d353` | Level 4: High contributions |

### Typography

- **Font Family**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Helvetica, Arial, sans-serif, "Apple Color Emoji", "Segoe UI Emoji"` (GitHub's native font stack) for optimal readability and developer-first feel.

---

## ✨ Features

- **GitHub Sidebar**: Left-aligned avatar, bio, location, email, personal website, and social connections (LinkedIn, Twitter/X, etc.).
- **Interactive Overview**: Layout of pinned repository cards, dynamic contribution heatmap grid, and README about-me view.
- **Repository Cards**: Display repositories with language dots (color-coded by language like JavaScript, Python, TypeScript, etc.) and star count.
- **Dynamic Contribution Heatmap**: Generates a 53-week contribution calendar via pure CSS Grid + DOM with realistic distribution.
- **README Markdown Engine**: A custom client-side parser inside `script.js` to render the user's details as a README file.
- **Responsive Layout**: Fluid double-column design collapsing cleanly to a unified mobile view.

---

## ⚙️ Customisation

All customization is powered by the `PORTFOLIO` object inside `script.js` which matches the dynamic parameters generated by the CareerPilot profile database.
